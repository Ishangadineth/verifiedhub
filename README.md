# Verified Hub

A modern, premium gateway for digital trust and scam awareness.

## Tech Stack
- Next.js 15 (App Router)
- React
- Vanilla CSS
- Lucide React (Icons)

## Features
- Multi-language support (Sinhala, Tamil, English)
- Digital Scam Awareness (Phishing, WhatsApp, Investment Fraud)
- Real-world Sri Lankan Case Studies
- Security Checklist for Safe Browsing
- Anonymous & Public Interest focused

## Deployment
Automated deployment via [Vercel](https://vercel.com).

## Mission
This project is a community-driven initiative focused on digital safety and scam prevention. It is maintained anonymously to protect the identity of contributors while serving the public interest.
